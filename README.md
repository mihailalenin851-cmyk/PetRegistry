# PetRegistry
Децентрализованный реестр для регистрации и учета домашних животных на базе смарт-контракта Solidity.

Проект разработан для безопасного ведения базы данных питомцев в блокчейне, связывания их с уникальными адресами владельцев и управления правами собственности.
Добавление питомца в систему с указанием его характеристик.
 Быстрый поиск хозяина животного по ID.
 Возможность официально передать питомца на другой адрес.

Как запустить локально
1. Клонируйте репозиторий:
   `git clone https://github.com`
2. Установите зависимости: `npm install`
3. Скомпилируйте контракт: `npx hardhat compile`
